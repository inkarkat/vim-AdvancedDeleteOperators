USAGE					       *AdvancedDeleteOperators-usage*
							 *c<Space>* *d<Space>*
c<Space>{motion}	Delete text that {motion} moves over, reduce any
			remaining whitespace to a single space, and start
			insert.
d<Space>{motion}	Delete text that {motion} moves over and reduce any
			remaining whitespace to a single space.
							       *c<BS>* *d<BS>*
c<BS>{motion}		Delete text that {motion} moves over, remove any
			remaining whitespace, and start insert.
d<BS>{motion}		Delete text that {motion} moves over and remove any
			remaining whitespace.

INSTALLATION				*AdvancedDeleteOperators-installation*

The code is hosted in a Git repo at
    https://github.com/inkarkat/vim-AdvancedDeleteOperators
You can use your favorite plugin manager, or "git clone" into a directory used
for Vim |packages|. Releases are on the "stable" branch, the latest unstable
development snapshot on "master".

This script is also packaged as a |vimball|. If you have the "gunzip"
decompressor in your PATH, simply edit the *.vmb.gz package in Vim; otherwise,
decompress the archive first, e.g. using WinZip. Inside Vim, install by
sourcing the vimball or via the |:UseVimball| command. >
    vim AdvancedDeleteOperators*.vmb.gz
    :so %
To uninstall, use the |:RmVimball| command.

DEPENDENCIES				*AdvancedDeleteOperators-dependencies*
